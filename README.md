# Huitot Dashboard

Mini project được làm trong môn HCI - Human Interaction

## Miêu tả

> Yêu cầu làm ra 1 mini project giống startup. Bất kì ý tưởng gì. 
> Đây là Dashboard UI quản lý toàn bộ tài khoản, dây hụi v.v.

## Tính năng

- Chỉ số truy cập, giao dịch
- Quản lý tài khoản
- Quản lý báo cáo
- Quản lý giao dịch
